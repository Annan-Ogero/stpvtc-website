## TIPS
- Use real photos from the college wherever possible — they build trust
- For project screenshots: open the student project in Chrome,
  press F12 > Device toolbar > set to 1280x720 > screenshot
- Free stock photos if needed: unsplash.com (search: "student computer",
  "coding Kenya", "classroom Africa", "web development")
- Compress all images before uploading: squoosh.app (aim for under 200KB each)
